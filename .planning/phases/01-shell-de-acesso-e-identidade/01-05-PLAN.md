## Gap Context

**Gap 1: Icon Serialization Error (Blocker)**
- Root cause: Lucide icons are React components being passed from Server Components to Client Components via props
- Current pattern: `MODULES` config stores actual React components in `icon` property
- Server Component pages pass `icon={mod.icon}` to Client Component `ModulePlaceholderPage`
- React cannot serialize component references across the Server/Client boundary

**Gap 2: Missing Cliente Profile (Major)**
- Root cause: 'cliente' profile was never added to `ProfileKey` type or profile configurations
- Current profiles: admin, commercial, partner, operations
- Missing: cliente with label "Cliente"

</context>
