- Each folder must have `entityStatus` values like `"Em andamento"`, `"Concluido"`, `"Aguardando"` and `entityResponsible` with real-sounding names.
- File sizes in bytes (range 50000 to 15000000). Dates in ISO format `"2025-01-15"`.

**Helpers:**
- `export function getDriveFolders(section: DriveSection): DriveFolder[]` -- returns folders filtered by section, sorted alphabetically by name
- `export function getDriveFolderById(folderId: string): DriveFolder | undefined`
- `export function getFilesInFolder(folderId: string, subfolderId?: string | null): DriveFile[]` -- returns files for a folder, optionally filtered by subfolder
- `export function formatFileSize(bytes: number): string` -- returns human-readable size like `"2.4 MB"` or `"156.0 KB"`
- `export function searchDrive(query: string, section: DriveSection): { folders: DriveFolder[]; filesWithFolder: Array<{ file: DriveFile; folder: DriveFolder }> }` -- searches folder names and file names, returns matching folders and files with their parent folder

Use `toLocaleDateString("pt-BR")` comment note for date display. Do NOT import or use any UI components in this file -- it is a pure data module.
